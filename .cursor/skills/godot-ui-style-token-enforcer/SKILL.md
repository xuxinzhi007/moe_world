# UI 风格规范执行

## 目标

让所有页面遵循同一套样式标准，提高一致性与可维护性。

## 适用范围

- `Scenes/ui/` 所有页面与弹层
- `Scripts/meta/ui_theme.gd` 相关样式调用

## 规范执行流程

1. 先对照文档：`docs/FLOW_UI_DESIGN_BLUEPRINT.md`
2. 抽取页面当前样式差异（字号、间距、按钮高度、圆角）。
3. 优先复用主题函数，减少每页单独覆写。
4. 新页面按同一 token 落地，旧页面逐步收敛。

## 默认 token（可按设计稿更新）

- 标题字号：`22~34`
- 正文字号：`14~16`
- 小字：`11~13`
- 主按钮高：`56~64`
- 次按钮高：`40~48`
- 圆角：`14~24`

## 输出格式

- 发现的样式不一致项
- 本次统一策略
- 涉及场景/脚本
- 可视化验收建议（PC/移动端）
