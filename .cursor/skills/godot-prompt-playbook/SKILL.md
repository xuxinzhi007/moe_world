# Godot 提示词模板手册

## 目标

给 AI 使用统一任务模板，减少歧义，提高一次成功率。

## 使用方式

1. 先选择任务类型模板。
2. 填写“目标/约束/验收”三部分。
3. 明确必须改动的文件路径与不允许改动范围。
4. 要求输出“变更说明 + 验证步骤”。

## 模板入口

- 详细模板见 [prompts.md](prompts.md)

## 关键约束建议（每次都写）

- 仅做最小必要改动。
- 保持中文文案不变（除非明确要求）。
- 同时验证 PC 与移动端关键路径。
- 如涉及世界/试炼，双场景都要回归。
