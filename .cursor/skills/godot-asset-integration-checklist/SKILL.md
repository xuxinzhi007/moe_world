# 资源接入检查清单

## 目标

让新资源稳定接入，不破坏现有逻辑和性能。

## 适用范围

- 角色立绘、技能特效、UI 图标、地形装饰、BGM/SFX。

## 执行步骤

1. 确认资源路径和命名，避免后续动态加载失败。
2. 优先用固定路径/预加载方式接入关键资源。
3. 校正显示参数：缩放、锚点、偏移、z_index。
4. 校验触发逻辑：何时播放、何时停止、是否重复播放。
5. 完成后做场景内实测（大厅、世界、试炼至少一处）。

## 重点文件

- `Assets/`
- `Scenes/`、`Scenes/ui/`
- `Scripts/combat/`、`Scripts/meta/`、`Scripts/autoload/game_audio.gd`
- `Scripts/world/world_scene.gd`

## 输出格式

- 资源接入清单（旧资源 -> 新资源）
- 调整参数清单（scale/offset/z）
- 验证结果（可见性、播放、性能）

## 常见问题提示

- 资源显示异常：先看缩放、偏移、层级。
- 资源不加载：先看路径与文件名。
- 音频混乱：先看播放时机与重复触发。
