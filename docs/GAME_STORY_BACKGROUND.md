# Moe World 世界背景设定（V0.1）

> 目标：给当前“大厅-世界-试炼-成长”玩法提供统一叙事动机，先轻设定、可迭代，不增加重剧情负担。

---

## 1. 世界一句话

- 在“雾潮纪元”后的萌境大陆，裂隙持续泄露异化能量，玩家作为巡游者在探索与试炼中回收晶核，修复失衡生态区。

## 2. 时间与环境

- **雾潮纪元**：七年前，天空裂开“静默缝隙”，怪潮与污染随之出现。
- **四域失衡**：林地、潮滩、荒野、遗迹四个生态区陆续受影响。
- **幸存据点**：玩家当前所在大厅是“回响前哨”，负责补给、整备与联机集结。

## 3. 玩家身份（与现有玩法对齐）

- **身份**：萌境巡游者（可理解为自由佣兵 + 修复者）。
- **主循环职责**：
  - 进入大世界清理怪物，收集材料；
  - 进入试炼压制怪潮，获得评级与稀有晶核；
  - 回到大厅强化成长与装备，再次出发。

## 4. 现有系统叙事映射

- **史莱姆凝胶**：雾潮低阶污染物冷凝后的稳定胶质（常见材料）。
- **试炼晶核**：试炼场高压环境下凝聚的战斗核心（稀有材料）。
- **林地树脂**：被污染林地析出的修复介质（成长与锻造通用材料）。
- **古旧骨片**：旧时代守卫残骸中的硬质碎片（后续可扩展为高阶强化材料）。

## 5. 版本阶段叙事目标（轻量）

- **P0-P1（当前）**：建立“为什么战斗”的基础解释，确保玩家知道自己在修复世界。
- **P2**：为每个区域补一条 3-5 句短故事（进入时气泡提示即可）。
- **P3**：增加区域首领/事件文案，让掉落与剧情线索绑定。

## 6. 文案接入建议（最小改动）

- 大厅“公告/通知”入口：加入世界观导读（已接入）。
- 试炼结算：后续可在评级行后补一句“回收进度”说明。
- 新区域首次进入：展示 1 条一句话背景，不打断操作。

## 7. 风格约束

- 叙事风格：轻幻想 + 可爱冒险，不走沉重黑暗路线。
- 文案长度：单屏可读完（移动端优先），一段不超过 3 行。
- 术语统一：雾潮、裂隙、晶核、巡游者、回响前哨。
