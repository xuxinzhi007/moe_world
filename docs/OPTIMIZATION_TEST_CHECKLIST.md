### C. 大世界性能与输入

- [ ] 顶栏“在线人数”显示稳定（不会每帧抖动/闪动），人数变化时能正常更新。
- [ ] 接触伤害判定在多怪围攻时更顺滑（加入预筛逻辑后无误伤/漏伤）。
- [ ] 玩家头顶 UI（名字/血条/经验条）在移动战斗中位置稳定、刷新正常。

### D. 移动 UI 控件性能与清理

- [ ] 技能按钮冷却绘制不再每帧重绘，显示仍准确。
- [ ] 切换场景后无输入异常、无重复信号引发的行为异常。

### E. 本轮新增修复（本次请重点补测）

- [ ] 大世界 `world_scene.gd` 在退出场景后，`size_changed` 与 `CharacterBuild.build_changed` 信号会正确断开（反复进出世界无重复回调）。
- [ ] 试炼 `survivor_arena.gd` 的怪物接触伤害判定已加预筛（高怪量时更稳，无漏伤/误伤）。
- [ ] 玩家 `player.gd` 交互时不再每帧查询 `world_map_open` 分组（地图开关与交互仍正确）。
- [ ] 怪物 `world_monster.gd` 已将主要距离判定改为平方判定（大规模怪物时追踪/冲刺逻辑表现正常）。

### F. P0 闭环修复（本次新增）

- [ ] 大世界死亡后会弹出结算层（“原地恢复 / 返回大厅”），且期间输入被锁定。
- [ ] 点击“原地恢复”后：血量回满、角色恢复可控、结算层正常关闭。
- [ ] 点击“返回大厅”后：场景切换正常，不出现重复切换或卡在黑屏。
- [ ] 云端模式离场时，`world_scene` 相关云端信号会断开；重复进出不会出现重复回调。

### G. 玩法闭环增强（本次新增）

- [ ] 试炼结算后会发放材料（`史莱姆凝胶`、`试炼晶核`），返回大世界后背包仍可见。
- [ ] 武器商店改为材料购买：材料不足时按钮不可购买，并提示收集材料。
- [ ] 材料足够时可成功购买并自动装备，背包对应材料数量会正确扣减。
- [ ] 试炼奖励与评级挂钩：S/A/B/C/D 档返回后材料收益有明显差异（S > A > B > C > D）。
- [ ] 大世界怪物掉落新增材料（`林地树脂`、`古旧骨片`），背包可正常显示。
- [ ] 成长面板可消耗材料兑换成长点（不足时提示、足够时点数增加并扣材料）。

## 三、建议你本次冒烟的执行顺序（10-15 分钟）

### 1) 启动与场景流

- [ ] 从大厅进入大世界，停留 1 分钟观察控制台是否有新增报错。
- [ ] 从大世界进入试炼，战斗 2-3 分钟后返回大世界。
- [ ] 再次进入试炼，重复一次，确认不会第二次进入就异常。
- [ ] 大世界被怪物击倒后，验证“原地恢复”和“返回大厅”两条路径都可正常走通。

### 2) 职业战斗回归（重点）

- [ ] 战士：普攻命中反馈、震屏、伤害正常。
- [ ] 法师：AOE 命中范围和伤害正常。
- [ ] 牧师：普攻有圣光攻击伤害，同时保留治疗反馈。
- [ ] 弓手：箭矢命中后反馈正常。

### 3) 高负载场景

- [ ] 试炼怪物较多时，操作响应是否比之前流畅。
- [ ] 大世界聚怪时接触伤害是否稳定、不卡顿、不瞬间异常掉血。

### 4) 输入系统

- [ ] 移动端控件正常；切换到键盘后可无缝控制。
