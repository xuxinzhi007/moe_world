# 萌社区开发助手 - 主入口

## 📋 项目概述
- **项目名称**: 萌社区 (Moe World)
- **项目类型**: 2D 社区社交游戏
- **游戏引擎**: Godot 4.4
- **脚本语言**: GDScript
- **项目路径**: `d:\godot_data\moe_world`

---

## 🧭 子技能索引

### 需要了解项目基础信息
**调用**: `moe-community-core`
- 项目概述、技术栈、版本
- Godot 可执行文件路径
- 核心开发规则
- Godot 4.4 与 4.5 兼容性说明

**何时调用**: 开始任何开发工作前

---

### 需要修改场景或节点
**调用**: `moe-community-nodes`
- 所有场景的节点树结构
- 节点引用路径
- 动态生成的节点说明

**何时调用**: 修改任何场景或节点前

---

### 需要编写或修改脚本
**调用**: `moe-community-scripts`
- 所有脚本的作用和挂载位置
- 代码规范和命名规范
- 核心函数和信号说明

**何时调用**: 编写或修改任何脚本前

---

### 需要对接或修改 API
**调用**: `moe-community-api`
- 后端 API (Go-Zero) 接口文档
- 本地 AI (Ollama) 对接配置
- 启动和验证命令

**何时调用**: 对接或修改任何 API 相关代码前

---

### 需要检查代码或遇到错误
**调用**: `moe-community-check`
- 常见错误和修复方法
- 验证和调试方法
- 代码检查清单
- 常见问题解答

**何时调用**: 写完代码或遇到错误时

---

## 🚀 开发流程

### 标准开发流程
1. **先调用 `moe-community` (本技能)** - 了解项目
2. **根据需要调用对应子技能** - 获取具体信息
3. **修改代码** - 按照规范开发
4. **最后调用 `moe-community-check`** - 检查代码
